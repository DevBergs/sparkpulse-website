## Adding screenshots

Place PNG files in `screenshots/`:
- `screenshots/dashboard.png`
- `screenshots/focus-timer.png`
- `screenshots/activations.png`

Capture from Xcode Simulator (iPhone 15 Pro Max) → File → Save Screen.

Then push:
```bash
git add screenshots/
git commit -m "Add screenshots"
git push
```
